# Vue Pin Input (Vue 3)

Vue Pin Input is a Vue 3 input component for pins, otp etc.

## Installation


```bash
yarn v-pin-input
# or
npm v-pin-input
```

## Usage
```ts
import { createApp } from 'vue'
import App from './App.vue'

import PinInput from 'v-pin-input'

const app = createApp(App)

app.use(PinInput)
app.mount('#app')
```
